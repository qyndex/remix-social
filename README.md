# Remix Social

Social network feed with post composition, like reactions via useFetcher, user profiles with follow action, and full ErrorBoundary coverage.

**Stack:** Remix | **Complexity:** complex
**Tags:** `remix`, `react`, `social`, `feed`, `profiles`, `reactions`, `fetcher`, `typescript`

## Quick Start

```bash
npm install
npm run dev
```

## Docker

```bash
docker compose up --build
```
